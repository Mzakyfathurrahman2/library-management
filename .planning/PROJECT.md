# Project: Library Management System

## Vision
A web-based digital solution for university libraries to replace manual spreadsheet management. The system serves two main user roles: Students and Librarians.

## Goals
- Digitally track book inventory.
- Automate book borrowing and return processes.
- Manage student member registration.
- Provide a searchable catalog for students.
- Maintain historical records of all transactions.

## User Roles
### Student
- Search for books in the catalog.
- Check book availability.
- Borrow books (within limits).
- View personal borrowing history.

### Librarian
- Add/Edit/Remove books from inventory.
- Register new student members.
- Process book returns.
- Monitor overdue books and borrowing history.

## Tech Stack
- **Frontend:** React with TypeScript.
- **Backend:** Node.js with Express.
- **Styling:** Vanilla CSS.
- **Database:** (To be determined during research/planning, likely PostgreSQL or MongoDB).
